# Opportunities Hub - Student Project

A mobile-first web application that displays student information and fetches posts from the Opportunities Hub WordPress API.

## Features

- **Student Information Page**: Displays all team members with their names and registration numbers
- **Posts Feed**: Fetches and displays posts from the WordPress REST API
- **Mobile-First Design**: Optimized for mobile devices and web view apps
- **Modern UI**: Clean, elegant interface with smooth interactions

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## API Endpoint

The application fetches posts from:
`https://opportunitieshub.ng/wp-json/wp/v2/posts`

## Technologies

- React 18
- Vite
- Modern CSS with CSS Variables
- Mobile-First Responsive Design
